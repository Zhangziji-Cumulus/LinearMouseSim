# 打包为 Windows EXE 文件计划

## 目标
将 LinearMouseSim 打包为独立的 Windows EXE 文件，无需安装 Python 即可运行。

## 项目依赖
- keyboard==0.13.5
- pillow==12.3.0
- pyvjoy==1.0.1
- vgamepad==0.1.0

## 打包配置
- **打包模式**：单文件模式（--onefile）
- **程序图标**：自定义图标（需要 .ico 文件）
- **配置文件位置**：EXE 同目录

## 新增功能：ViGEmBus 驱动自动检测

### 检测方法
1. 检查注册表：`HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\ViGEmBus`
2. 检查驱动文件：`C:\Windows\System32\drivers\ViGEmBus.sys`

### 用户交互流程
1. 程序启动时检测 ViGEmBus 驱动
2. 如果未安装，弹出提示对话框
3. 提供"下载驱动"按钮，自动打开浏览器下载
4. 提供"继续使用"按钮，进入模拟模式（无虚拟手柄输出）

### ViGEmBus 下载地址
https://github.com/ViGEm/ViGEmBus/releases

## 打包步骤

### 1. 创建 requirements.txt
```
keyboard==0.13.5
pillow==12.3.0
pyvjoy==1.0.1
vgamepad==0.1.0
pyinstaller
```

### 2. 创建 PyInstaller 打包配置文件 (LinearMouseSim.spec)
- 入口文件：main.py
- 模式：单文件（--onefile）
- 隐藏控制台窗口（--noconsole）
- 自定义图标
- 包含所有子模块

### 3. 执行打包命令
```bash
pip install pyinstaller
pyinstaller LinearMouseSim.spec
```

### 4. 打包输出
- 输出目录：`dist/LinearMouseSim.exe`
- 文件大小：约 10-20MB

## 注意事项
1. keyboard 模块需要管理员权限运行
2. 单文件模式启动较慢（需要解压临时文件）
3. 用户需要安装 ViGEmBus 驱动才能使用虚拟手柄功能

## 验证方式
1. 运行 `dist/LinearMouseSim.exe`
2. 确认程序正常启动
3. 测试 ViGEmBus 驱动检测功能
4. 测试所有功能正常工作
