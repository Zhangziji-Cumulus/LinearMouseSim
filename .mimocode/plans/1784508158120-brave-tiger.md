# 光标锁定位置移到屏幕顶部中间

## 改动

`core/state_machine.py` — 将光标锁定的 Y 坐标从屏幕中心改为屏幕顶部（Y=1）。

当前代码：
```python
self.center_x, self.center_y = get_screen_center()
```

改为：
```python
self.center_x, _ = get_screen_center()
self.center_y = 1  # 锁定在屏幕顶部中间，防止遮挡游戏视野
```

`_cursor_lock_loop` 和 `turn_on` 中所有使用 `self.center_x, self.center_y` 的地方无需修改，因为变量名不变。

## 文件

| 文件 | 改动 |
|------|------|
| `core/state_machine.py` | 第 20 行：`center_y` 从屏幕中心改为 1 |

## 验证

启动程序，按 O 开启模拟 → 光标应隐藏在屏幕顶部中间位置。
