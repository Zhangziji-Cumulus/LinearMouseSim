# LinearMouseSim v0.0.1

首个测试版本发布 - 鼠标模拟方向盘输入工具

## 简介

LinearMouseSim 是一个 Windows 桌面应用，将鼠标移动转换为线性方向盘输入，专为尘埃拉力赛 2.0 等赛车游戏设计。无需购买昂贵的方向盘设备，使用鼠标即可获得线性转向体验。

## 功能特性

### 核心功能
- **鼠标模拟方向盘** - 将鼠标水平移动转换为方向盘角度，支持最高 540° 转向
- **虚拟 Xbox 360 手柄** - 使用 ViGEmBus 模拟标准手柄，游戏原生支持，无需额外配置
- **油门/刹车控制** - 右键按住为油门，左键按住为刹车
- **智能辅助回中** - 三状态机设计，解决大角度回打时的卡顿问题

### 参数配置
- **高度可配置** - 灵敏度、死区、最大角度、曲线类型等 20+ 参数
- **自定义预设** - 保存/加载/切换多套配置方案
- **DPI 自适应** - 自动适配不同鼠标 DPI（默认 800 DPI 基准）

### 界面与交互
- **实时方向盘动画** - 可视化显示当前转向角度
- **全局快捷键** - 可自定义的热键系统
- **OSD 叠加显示** - 实时显示灵敏度变化等信息
- **系统托盘** - 最小化到托盘，支持后台运行

## 安装说明

### 系统要求
- Windows 10/11
- ViGEmBus 驱动（首次运行会自动提示安装）

### 安装步骤
1. 从 [Releases](https://github.com/Zhangziji-Cumulus/LinearMouseSim/releases) 下载 `LinearMouseSim.exe`
2. 首次运行时，程序会检测 ViGEmBus 驱动，如未安装会弹窗提示下载
3. 安装 ViGEmBus 驱动后重启程序即可使用

## 使用说明

### 基本操作
1. 启动程序后，按 `}` 键开启/关闭模拟
2. 开启后，鼠标左右移动控制方向盘转向
3. 右键按住为油门，左键按住为刹车
4. 按 `0` 键重置方向盘到中心位置

### 默认快捷键
| 按键 | 功能 |
|------|------|
| `}` | 开启/关闭模拟 |
| `+` | 增加灵敏度 |
| `-` | 降低灵敏度 |
| `0` | 重置方向盘 |
| `F5/F6/F7` | 切换灵敏度预设 1/2/3 |
| `Shift` (按住) | 临时半灵敏度 |
| `Ctrl` + 滚轮 | 调节灵敏度 |

### 参数调节
程序界面提供 5 个设置标签页：
1. **基本设置** - 灵敏度、平滑度、死区、最大角度、DPI
2. **辅助回中** - 回打检测、归中参数
3. **方向盘回中** - 自动回中模式和速度
4. **映射参数** - 像素到角度的映射
5. **快捷键** - 自定义热键

## 构建说明

### 依赖安装
```bash
pip install -r requirements.txt
```

### 打包为 exe
```bash
pyinstaller LinearMouseSim.spec --clean
```
生成文件位于 `dist/LinearMouseSim.exe`
